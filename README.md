# Проект: Место - [Посмотреть](https://owlspace.github.io/mesto/src/index.html)

Mesto - интерактивная страница, где можно добавлять фотографии, удалять их и ставить лайки.

### Основной функционал:

* Редактирование профиля

### Технологии:

* HTML5
* CCS3
* JavaScript
* Адаптивная верстка
* Методология БЭМ
